#Turning words into phone numbers

Given an expression, you should turn that into a phone number. These are the number values related to the letters:

"A, B, C" -> "2"
"D, E, F" -> "3"
"G, H, I" -> "4"
"J, K, L" -> "5"
"M, N, O" -> "6"
"P, Q, R, S" -> "7"
"T, U, V" -> "8"
"W, X, Y, Z" -> "9"


For "1" you should return "1"

```
1-HOME-SWEET-HOME
MY-MISERABLE-JOB
```

Result
```
1-4663-79338-4663
69-647372253-562
```
